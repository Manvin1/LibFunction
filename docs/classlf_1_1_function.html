Detailed Description

Classe que representa uma função matemática.

Esta classe representa uma função matemática, oferecendo uma interface para a sua definição e evaluação de seus valores.

◆ Resolve() [1/2]

double lf::Function::Resolve ( )

Evaluir a função a fim de obter o seu valor.

Esta função deve ser a usada quando não se tem alguma variável. Contudo, caso alguma seja encontrada durante a resolução, o valor inicial 0.0 será usado. Esta função pode afetar a performance.
